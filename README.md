# Image Generation using Finetuned Diffusion based Model

This repository contains all the code for our finetuned Stable Diffusion model for image synthesis.

## Running the project
1. Firstly, clone this repository using
   
   ```bash
    git clone https://github.com/Pranish-Nepal/Image-Generation-using-Finetuned-Diffusion-based-Model.git
   ```
   on your machine.
   
3. To install dependencies of JavaScript, run `npm install` in terminal. This will install all the dependencies included in package.json file.
4. Run `pip install -r requirements.txt` to install all the required Python dependencies.
  
   To pack the dependencies, use `pip freeze > requirements.txt`.
